# 🏥 Healthcare Appointment Management System

A web-based Healthcare Appointment Management System developed using Flask and SQLite.

## Features

- Patient Registration & Login
- Admin Dashboard
- Doctor Dashboard
- Book Appointment
- View Doctors
- View Patients
- View Appointments
- Add/Edit/Delete Doctors
- Responsive UI using Bootstrap

## Technologies Used

- Python
- Flask
- SQLite
- HTML5
- CSS3
- Bootstrap
- JavaScript
